How to use this application

  • Step 1: Input data: There are 2 ways to input your data
    1. Typing your scores in text area
      • Both Integer or real numbers are OK.
      • Separate each score with space.
      • Don't use Comma or any alphabets.
      • You can type scores in a line as long as you want or press 'enter' for a new line.
      • A star ( * ) at the beginning of the line means this line is ignored (It's a comment).
    2. You can type scores in any editor (NotePad) or spreadsheat (Excel) following the rule as mention in 1. then select all data(Ctr+a) copy(Ctr+c) and paste (Ctr+ v) into this text area.
      WARNING:**The maximum number of data: 3000.**
  • Step 2:Data Caculation
    When data had already been in text area, clicking on 'Calculate' button will launch the program to compute the statistical values. Sorted scores, frequencies T scores and grades being displayed in the table below the web page.
  • Step 3: Changing grade.
    'A,B,B+,C,C+,D,D+,F' is a default grade interval. To change the grade interval, selecting any specific range as shown in combo box. The new grades will be calculated automatically. (without clicking 'Calculate' button)
  • Step 4: Printing the result.
    Just clicking on 'Print result' button, the output result will be sent to the printer.
